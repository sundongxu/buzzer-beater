# 🏀 BuzzerBeater - 产品设计文档

## 一、产品定位

### 核心理念
**让篮球更有趣，让球迷更懂球，让打球更容易**

### 与虎扑的差异
| 维度 | 虎扑 | BuzzerBeater |
|------|------|--------------|
| **内容形式** | 以论坛帖子为主 | 短视频+图文+数据可视化 |
| **数据展示** | 传统表格 | 动态图表、热力图、3D可视化 |
| **社交方式** | 单向讨论 | 多元互动（连麦、语音房、实时聊天） |
| **约球功能** | 无 | 完整的约球系统 |
| **用户体验** | 信息密集 | 简洁现代、视觉友好 |
| **互动玩法** | 较少 | 竞猜、Fantasy、成就系统 |

#### 内容类型
1. **图文帖子**
   - 支持最多9张图片
   - 图片可点击放大查看
   - 支持长图和多图横滑

2. **短视频**
   - 15秒-3分钟
   - 自动播放（滑到可视区域）
   - 双击点赞
   - 下滑切换下一个视频

3. **数据卡片**
   - 比赛数据可视化
   - 球员对比图表
   - 赛季趋势分析
   - 可交互的动态图表

4. **投票帖**
   - 多选/单选投票
   - 实时显示投票结果
   - 饼图展示占比

5. **长文章**
   - 深度分析文章
   - 支持Markdown格式
   - 目录导航
   - 代码块展示战术

#### 交互设计

**手势操作**
- 👆 **点击卡片**: 进入详情页
- 👆👆 **双击**: 快速点赞
- 👈 **左滑**: 显示"收藏"按钮
- 👉 **右滑**: 显示"分享"按钮
- ⬆️ **长按**: 预览用户主页
- ↕️ **上下滑动**: 浏览内容

**实时更新**
- 顶部显示"有X条新内容"提示条
- 点击自动滚动到顶部加载新内容
- 实时比赛分数自动更新（无需刷新）

## 三、用户使用场景

### 场景1: 上班通勤看比赛
**小明的一天早上**
1. 打开App，首页自动显示昨晚NBA比赛战报
2. 刷到一条"库里昨晚50分"的短视频集锦
3. 双击点赞，顺手转发到微信
4. 点进比赛详情，看投篮热图
5. 在评论区和其他球迷讨论

**时长**: 10-15分钟

### 场景2: 午休约球
**小张想下班打球**
1. 打开"约球"页面
2. 看到附近有3个球场
3. 选择最近的朝阳公园
4. 发现今晚19:00有人组局
5. 点击"参加"，等待确认
6. 在讨论区问是否需要带球
7. 收到导航提醒，下班后直接导航过去

**时长**: 3-5分钟

### 场景3: 晚上看球社交
**晚上8点，湖人vs勇士**
1. 打开App，顶部自动显示正在直播的比赛
2. 点进比赛详情，切换到聊天室
3. 开启弹幕模式
4. 边看边发送评论和表情
5. 看到精彩球，截屏分享到社区
6. 参与"本场MVP"投票

**时长**: 整场比赛2-3小时

### 场景4: 闲暇时间刷内容
**周末下午在家**
1. 打开App，上下滑动浏览Feed流
2. 看到一篇深度战术分析文章，点进去阅读
3. 收藏这篇文章
4. 继续刷，看到朋友发的约球照片
5. 点赞评论"下次叫我"
6. 刷到一条搞笑视频，分享到群里

**时长**: 30-60分钟

### 场景5: 参与竞猜
**比赛前的准备**
1. 打开竞猜页面
2. 看到今晚湖人vs勇士的竞猜
3. 查看两队数据对比
4. 看AI推荐建议
5. 投注200金币给湖人
6. 等待比赛结束揭晓结果

**时长**: 5分钟

## 四、视觉设计风格

### 4.1 色彩方案

**主色调**
- 🔴 **热情橙红**: #FF5722 - 主要行动按钮、强调元素
- 🔵 **深邃蓝**: #1976D2 - 链接、次要按钮
- ⚫ **深色背景**: #121212 - 深色模式主背景
- ⚪ **浅色背景**: #FFFFFF - 浅色模式主背景

- 🟢 **成功绿**: #4CAF50 - 成功提示
- 🟡 **警告黄**: #FFC107 - 警告提示
- 🔴 **错误红**: #F44336 - 错误提示
- 🟣 **紫色**: #9C27B0 - 特殊标识

**NBA球队色**
- 每个球队用官方配色
- 湖人紫金、勇士蓝黄、凯尔特人绿色等

### 4.2 字体系统

- 标题: 思源黑体 Bold
- 正文: 苹方/Roboto Regular
- 数字: DIN Alternate Bold

**字号规范**
- 大标题: 24-28sp
- 小标题: 18-20sp
- 正文: 14-16sp
- 辅助文字: 12sp

### 4.3 图标风格

- 线性图标为主
- 圆角设计
- 2px线宽
- 24×24px基础尺寸

### 4.4 动效设计

**页面转场**
- 淡入淡出 (300ms)

**交互反馈**
- 按钮按下缩放 (0.95)

- 加载骨架屏

**数据动画**
- 数字滚动效果
- 进度条动画
- 图表逐步渲染
