# BuzzerBeater Frontend

**让篮球更有趣，让球迷更懂球，让打球更容易**

## 技术栈

- Flutter 3.5+
- Dart 3.5+
- 支持平台：iOS、Android、Web

## 快速开始

### 安装依赖

```bash
flutter pub get
```

### 运行项目

#### iOS
```bash
flutter run -d ios
```

#### Android
```bash
flutter run -d android
```

#### Web
```bash
flutter run -d chrome
```

### 构建项目

#### iOS
```bash
flutter build ios
```

#### Android
```bash
flutter build apk
```

#### Web
```bash
flutter build web
```
